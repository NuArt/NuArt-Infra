# ADR-010: Dokumentace v Git markdown, ne dedikovaný nástroj

**Rozhodnutí:** Infra dokumentace (tohle repo) je markdown v Gitu, ne wiki/Notion/jiný nástroj.

**Proč:** Jednoduchost, verzování, čitelnost (i pro AI agenty), žádný další provozovaný nástroj.

**Důsledek:** Toto repo `nuart-infra` je zdroj pravdy o infrastruktuře; aktualizuje se commity
při změnách na serveru.
